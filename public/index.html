*  If 2020      *

*  were a poem  *

* * * * * * * * *

Oh, dear social media, canvas and medium to our most diverse insights and complaints… Those who walk the path of angry typing shall thrive in the end.

This page uses pieces of data to generate short free-verse poems. Each one is consisted of a random three-part combination:
- A tweet that mentions the current year;
- A relevant headline from The Guardian since last January;
- A sentence from Wikipedia 2020’s page.

Coded in Javascript and p5.js, the algorithm makes requests to each API using specific parameters which are returned as text. Not every poem makes sense, not every poem is good—but that’s the charming part about it.

Created by Marina Cardoso during her Computational Arts Masters at Goldsmiths University of London, 2020. You can see the source code here.
